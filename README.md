# Module 7 Week B — Core Skills Drill: Pre-Trained Pipelines & Metrics

This is the starter repo for the Module 7 Week B Core Skills Drill.

The full assignment guide is at <a href="https://levelup-applied-ai.github.io/aispire-14005-pages/modules/module-7/eee09191" target="_blank">the drill guide page</a> — read it first.

## Quick start

```bash
pip install -r requirements.txt
git checkout -b drill-7b-pretrained-pipelines
# Implement the TODOs in drill.py
pytest tests/ -v   # runs the autograder tests locally
```

## What's in this repo

- `drill.py` — your implementation file with five TODO functions.
- `tests/test_drill_7b.py` — the autograder tests. They will run automatically on every push via GitHub Actions.
- `requirements.txt` — Python dependencies (`transformers`, `torch` CPU build, `pandas`, `numpy`, `rouge-score`, `pytest`).
- `LICENSE` — proprietary educational license.

## Submission

When all tests pass: open a Pull Request from your `drill-7b-pretrained-pipelines` branch into `main`. Then paste the PR URL into TalentLMS → Module 7 → Core Skills Drill 7B.
